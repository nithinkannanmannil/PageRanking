Minimum Software Requirements
==============================
Java jdk version 1.5+

Google Chrome 64bit version 107.0.5304.62 or above


Minimum Hardware Requirements
==============================
10Mb+ internet connection


Operational Guide
====================
For Windows environment, run start.bat executable file.

For Linux environment, run start.sh executable file.

For changing the domain to PageRank, edit the executable file (start.bat or start.sh) used to start the program.

The domain should always end with a forward slash "/" eg. "https://www.gatech.edu/"
